# Report Format

The forensic report uses one shared structure:

## Token Details
- name
- address
- network
- launch date when available

## Market Activity
- price
- holders
- market cap
- liquidity
- 1H volume
- 1H change
- 24H volume
- 24H change
- 24H transactions

## Findings
Numbered findings derived from the strongest evidence.

## Conclusion
Three short sentences:
1. what the token should be treated as
2. what raw evidence supports that
3. what the safer next action is

## Case Report
- date
- case ID
- type
- reason
- X Layer (tx hash)
- View on Explorer

## Uniswap Plan
- suggested route
- planning note
- real Uniswap deep link
